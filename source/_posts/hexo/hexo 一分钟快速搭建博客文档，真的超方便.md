# 基础要求
> 环境：node
> 命令：[`hexo`](https://hexo.io/)（没有的话就安装：`npm install hexo-cli -g`）




# 1.项目初始化
```
hexo init 项目名
```

![](https://img-blog.csdnimg.cn/direct/e60d20f452274f97a5565d0bdbb30298.gif)
初始化完进如项目
```
cd blog
```



# 2.安装依赖
```
npm install
```

![](https://img-blog.csdnimg.cn/direct/973f25e106b64a14aee84984a86b2fad.gif)

# 3.运行项目
```
hexo server
```
