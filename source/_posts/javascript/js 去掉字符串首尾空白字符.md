> ##### trim()

```html
<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>Document</title>
</head>
<body>
	<script>
		var array="             666  ";
		
		console.log(array.trim()); // 去空字符
		console.log(array);
	</script>
</body>
</html>
```
