> 属性：**filter**

属性值：
![](https://img-blog.csdnimg.cn/20190926123520200.png)

> 语法：filter：brightness(17%)；/* 图片变暗17% 。*/


例如：
![](https://img-blog.csdnimg.cn/20190926123703457.png)
```css
	img{
	    filter: blur(10px);  /* 模糊 */
	  }
```
![](https://img-blog.csdnimg.cn/20190926123711393.png)
